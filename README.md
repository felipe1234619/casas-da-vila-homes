# Casas da Vila Homes — Site bilíngue base

Estrutura criada para VS Code + Vercel com:
- PT / EN
- header e footer globais por fragmentos
- CSS global premium
- motion discreto com reveal ao scroll
- páginas principais prontas para expansão

## Estrutura
- `/pt/` versão em português
- `/en/` versão em inglês
- `/components/` header/footer globais
- `/assets/css/global.css` sistema visual unificado
- `/assets/js/header-loader.js` injeção de header/footer
- `/assets/js/main.js` motion base

## Observação comercial importante
Os materiais fornecidos possuem divergências de preços, datas de entrega e percentual de reajuste no lançamento. Antes da publicação, consolide um documento-mestre comercial único para alimentar a página de investimento.
